# GitHub Repository Organizer

A REST API for organizing and managing your starred GitHub repositories.

## Setup
```bash
# Activate virtual environment
source bin/activate  # On Mac/Linux
# OR
bin\Scripts\activate  # On Windows

# Install dependencies
pip install -r requirements.txt

# Run the app
uvicorn main:app --reload
```
